# PU JIAJIE 时尚设计师官网

个人作品集网站，展示时尚设计师濮家捷的职业履历、作品集和核心能力。

## 网站结构

- **首页**：个人简介和核心定位
- **关于我**：教育背景和专业经历
- **职业履历**：工作经历和项目经验
- **作品集**：趋势企划、品牌设计和跨界设计作品
- **核心能力**：专业技能和资源优势
- **联系我**：联系方式和合作咨询

## 技术栈

- HTML5
- Tailwind CSS v3
- Font Awesome
- JavaScript

## 部署方式

可通过以下平台部署：
1. GitHub Pages
2. Vercel
3. Netlify
4. Surge

## 本地预览

```bash
# 使用Python HTTP服务器
python3 -m http.server 8000

# 或使用Node.js http-server
npx http-server -p 8000
```

然后在浏览器中访问 http://localhost:8000
